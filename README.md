# 塔科夫改枪王

用改枪码和你的朋友分享你的武器预设

**它能做什么？**

- 仅需一段代码, 就可以导入导出你改装的任何武器

- 完美融入游戏的导入导出按钮，以及完善的错误捕获系统

**如何使用？** 

- 在右侧的Release下载发布版本，解压后得到BepInEx文件夹，直接覆盖于游戏根目录即可。

**兼容性如何？**

- **与Fika的兼容性：** 完全兼容，我在实际测试中没有遇到任何问题。

- **与Fika.Co-op的兼容性：** 同上。

- **与Fika.Headless的兼容性：** 同上……好吧，考虑到它是一个纯客户端Mod并且不在战局内工作，理论上任何Mod都不会破坏它的运行。

**关于用法**

- 进入游戏后，编辑预设页面左上角的删除按钮下方会出现导入导出按钮，点击导出按钮，当前的改装方案会自动转换为改枪码，并复制到剪贴板
- 使用时，选择一把武器，然后点击导入按钮，即可将复制的改装方案一键导入
