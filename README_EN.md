## Tarkov Weapon Build Master

Share your weapon presets with your friends using a simple preset code!

**What does it do?**

- Import and export any of your customized weapon builds with just a single string of code.

- Seamlessly integrated Import/Export buttons within the native game UI, backed by a robust and comprehensive error-catching system.

**Installation**

- Download the latest version from the Releases tab on the right. Extract the archive and simply drop the BepInEx folder directly into your game's root directory.

**Compatibility**

- **Fika:**Fully compatible. I haven't encountered any issues during actual testing.

- **Fika.Co-op:** Same as above.

- **Fika.Headless:** Same as above... Well, considering this is a strictly client-side UI mod that doesn't even operate during active raids, theoretically, no other mod should break it.

**How to Use**

- Exporting: Once in the game, open the Edit Preset screen. You will see the new Import and Export buttons located right below the default Delete button. Click the Export button, and your current weapon build will be automatically converted into a preset code and copied directly to your clipboard.

- Importing: Select a base weapon on the workbench, ensure you have a preset code copied to your clipboard, and click the Import button. Your saved build will be assembled instantly with just one click!
